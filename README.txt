# Cobrinha do Abacate Legends

Versão mais avançada do projeto, com:
- mapas mais bonitos e mais variados;
- skins mais realistas e com efeitos;
- loja de upgrades;
- ranking local;
- seleção de fases;
- vários bosses (Tomatão, Pimentão e Abacatão);
- tutorial inicial;
- multiplayer, campanha, missões e conquistas.

## Como usar
1. Extraia o ZIP.
2. Abra o arquivo `index.html` no navegador.
3. Para usar como PWA/instalar, hospede os arquivos em um servidor local ou online.

## Arquivos

- `storage.js`

## Como publicar rápido
### GitHub Pages
1. Crie um repositório.
2. Envie todos os arquivos da pasta do jogo.
3. Vá em Settings > Pages.
4. Em Source, escolha a branch principal e a pasta raiz.
5. Abra o link gerado.

### Netlify
1. Entre no Netlify.
2. Arraste a pasta do jogo para "Deploy manually".
3. O site será publicado automaticamente.
