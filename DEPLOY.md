# 🚀 Como publicar o jogo

## Opção 1: GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos desta pasta.
3. Entre em **Settings > Pages**.
4. Em **Build and deployment**, escolha:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salve e aguarde o link ficar disponível.

## Opção 2: Netlify

1. Acesse o Netlify.
2. Clique em **Add new site**.
3. Arraste a pasta do jogo para o deploy.
4. O Netlify gera um link automaticamente.

## Opção 3: Vercel

1. Crie um projeto na Vercel.
2. Faça upload ou conecte o repositório.
3. Como é projeto estático, não precisa configurar build.
4. Publique.

## Antes de postar

Confira o arquivo `QA_CHECKLIST.md`.
